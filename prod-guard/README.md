# 快樂工具人: 正式區防呆小幫手 (Prod Guard)

這是一個專為「不想成為離職高手」的工程師開發的 Tampermonkey 腳本。它能在網頁上提供醒目的環境標識，並在正式區執行刪除操作時強制攔截確認。

> **「我的品牌是快樂工具人，我的使命是讓你不要手賤。」**

---

## 📸 畫面模擬 (文字版)

### 1. 環境識別標籤 (固定在左下角)
當進入設定網域時，會看到帶有 **快樂工具人 Logo** 的標籤：

* **正式區：** `[ 🔴 PROD 正式區 ]` (底色：紅色)
* **測試區：** `[ 🟠 STG 測試區 ]` (底色：橘色)
* **開發區：** `[ 🔵 DEV 開發區 ]` (底色：青色)

### 2. 正式區刪除確認視窗
在正式區點擊刪除按鈕時，會優先攔截並跳出：

```text
+--------------------------------------------------+
| ⚠️ 網頁顯示：                                      |
|                                                  |
| 【⚠️ 正式區操作確認】                               |
| 欲刪除：「重要客戶資料 (ID: 999)」                 |
|                                                  |
| 請輸入 DELETE 確認:                               |
| [ DELETE           ]                             |
|                                                  |
|        [ 取消 ]  [ 確定 ]                        |
+--------------------------------------------------+
