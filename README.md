## 🛠 安裝與使用

1. **先裝管理器**
   - 推薦 [Tampermonkey](https://www.tampermonkey.net/)，因為它最穩、最老牌。
2. **一鍵安裝**
   - 在上方的 **腳本列表** 中點擊「安裝」。
   - 跳出油猴頁面後點「安裝」，結束。

### ⚠️ 裝了沒反應？(權限設定)
現在瀏覽器比較龜毛（尤其是 Chrome/Edge 配合新版 Tampermonkey），如果裝完腳本沒動靜，請檢查這個開關：

1. 進入瀏覽器的 **「管理擴充功能」** 頁面。
2. 找到 **Tampermonkey** 的卡片/詳細資訊。
3. 確認 **「允許使用者指令碼」** (Allow User Scripts) 或 **「開發人員模式」** 已經**開啟**。
   > *沒開這個，瀏覽器會把腳本當作病毒擋在門外。*
