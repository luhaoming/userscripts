## 🛠 安裝與使用

1. **先裝管理器**
   - 推薦 [Tampermonkey](https://www.tampermonkey.net/)，因為它最穩、最老牌。
2. **一鍵安裝**
   - 在上方的 **腳本列表** 中點擊「安裝」。
   - 跳出油猴頁面後點「安裝」，結束。

## 🔄 自動更新
本 Repo 的腳本都帶有自動更新機制。我只要在 GitHub 推送新版本（並跳過 version 號碼），你的瀏覽器就會自動更新，不用理它。
