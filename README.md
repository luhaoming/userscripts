## 🛠 安裝與使用

1. **裝個管理器**
   - 推薦用 [Tampermonkey](https://www.tampermonkey.net/)，因為它最穩。
2. **點擊安裝**
   - 在上面的表格點「安裝」。
   - 跳出視窗後點「安裝」，結束。
